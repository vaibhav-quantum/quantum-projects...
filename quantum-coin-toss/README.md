# Quantum Coin Toss

## Objective
Simulate a fair coin using quantum superposition.

## Concepts Used
- Qubit
- Hadamard Gate
- Measurement

## Tools
- Python
- Qiskit

## Explanation
A Hadamard gate creates a superposition of |0⟩ and |1⟩.
Measurement collapses the state, producing true randomness.
